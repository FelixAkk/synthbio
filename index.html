Scrum iterations

On this page you will find our results per Scrum iteration. Note that the following demo's only demonstrate the front-end. Servlet responses have been replaced with stubs. If you want to use the program properly, please run it locally with your Tomcat server. For more information on how to set up a local version, see our wiki.

  • Scrum run 1 (JS tests)
    • Basic GUI (images will not be shown correctly in this demo - they will work on a local version)
    • Server connection status
    • List proteins
  • Scrum run 2 (JS tests)
    • Gate drag&drop into working space
    • Circuit modeling (drag gates & draw wires)
    • Stub protein selection
    • Stub gate removal (double-click)
  • Scrum run 3 (JS tests)
    • GUI polishing (fixed gate dropping aim; table sorting)
    • Protein selection
    • Input values specification
    • Circuit saving/loading
  • Work in progress: Scrum run 4 (JS tests)
    • GUI polishing (protein selection; task bar)
    • Advanced input values specification (tick width, low/high values)
    • Show output graph

For more information on each Scrum iteration and on the work done to the server, please refer to the wiki.
